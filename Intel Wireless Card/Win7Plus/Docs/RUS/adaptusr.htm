Настройки адаптера

На вкладке Дополнительно отображаются свойства устройства адаптера WiFi, установленного в компьютере.

Доступ

В меню "Дополнительно" утилиты подключения Intel® PROSet/Wireless WiFi щелкните Настройки адаптера. Щелкните вкладку Дополнительно.

Описание настроек адаптера WiFi

Имя

 Описание

Режим 802.11ac
(5 ГГц)

Стандарт 802.11ac основан на работе стандарта 802.11n. Режим 802.11ac обеспечивает до 867 Мбит/с (теоретически) путем повышения полосы пропускания канала до 80 МГц, а также добавляя модуляцию высокой плотности (256 QAM). Выберите Вкл. или Выкл. для установки режима работы адаптера WiFi 802.11ac. Значение "Вкл." установлено по умолчанию. Эта настройка применяется только к адаптерам с поддержкой частот 802.11ac.

Ширина канала 802.11n (2,4 ГГц)

Используется для установки ширины канала повышенной пропускной способности для увеличения производительности. Установите ширину канала в значения Авто или 20 МГц. Выберите 20 МГц, если использование каналов 802.11n ограничено. Эта настройка применяется только к адаптерам с поддержкой частот 802.11n.

ПРИМЕЧАНИЕ. Эта настройка не применяется к адаптерам Intel® Wireless WiFi Link 4965AGN (использует ширину канала только 20 МГц).

Ширина канала 802.11n (5,2 ГГц)

Используется для установки ширины канала повышенной пропускной способности для увеличения производительности. Установите ширину канала в значения Авто или 20 МГц. Выберите 20 МГц, если использование каналов 802.11n ограничено. Эта настройка применяется только к адаптерам с поддержкой частот 802.11n.

ПРИМЕЧАНИЕ. Эта настройка неприменима к следующим адаптерам:

  • Адаптер Intel® WiFi Link 1000
  • Адаптер Intel® Wireless WiFi Link 4965AGN

Режим 802.11n

Стандарт 802.11n основан на работе выпущенных ранее стандартов 802.11 и дополнен режимом множественного ввода/вывода (MIMO). Режим MIMO повышает пропускную способность для увеличения скорости передачи данных. Выберите Вкл. или Выкл. для установки режима работы адаптера WiFi 802.11n. Значение "Вкл." установлено по умолчанию. Эта настройка применяется только к адаптерам с поддержкой частот 802.11n.

ПРИМЕЧАНИЕ. Для обеспечения скоростей передачи более 54 Мб/c при подключениях 802.11n необходимо выбрать параметр защиты WPA2*-AES. Отсутствие защиты (Нет) может быть выбрано для включения настройки сети и поиска и устранения неисправностей.

Администратор может включить или отключить поддержку режима повышенной пропускной способности для уменьшения энергопотребления или вероятности конфликтов с другими частотами, а также других проблем совместимости.

Канал Ad Hoc 802.11b/g

Выберите Включено или Отключено.

Режим Ad Hoc QoS

Контроль качества обслуживания (Quality of Service (QoS)) в одноранговых сетях. Качество обслуживания обеспечивает выбор приоритета исходящего трафика точки доступа в беспроводную ЛС на основе его классификации трафика. WMM (Wifi MultiMedia) – это сертификация качества обслуживания (QoS) в сетях Wi-Fi (WFA). Если функция WMM включена, адаптер использует WMM для поддержки приоритета пометки и упорядочивания функций для сетей Wi-Fi.

  • WMM вкл. (по умолчанию)
  • WMM выкл.

Bluetooth® AMP

Включите или отключите Bluetooth® AMP. AMP является альтернативой MAC/PHY и использует стандарт 802.11 (Wi-Fi) в качестве высокоскоростного транспортного уровня. При отключении также отключается функция Bluetooth HS.

Режимы HT/VHT и их отключение

 Эта настройка позволяет выбрать режим HT (режим повышенной производительности), режим VHT (режим очень высокой производительности) или отключить оба режима. Режим HT поддерживает совместимость со стандартом 802.11n, а режим VHT — со стандартом 802.11ac.

Несовместимость с каналом

Эта настройка позволяет сообщать близлежащим точкам доступа, что данный адаптер WiFi несовместим с каналами шириной в 40 МГц в диапазоне 2,4 ГГц. По умолчанию она отключена, поэтому адаптер не отправляет данное предупреждение. Если точка доступа продолжает использовать каналы 40 МГц, адаптер WiFi также продолжит использование каналов 40 МГц, если для настройки ширины канала 802.11n (2,4 ГГц) задано значение "АВТО".

ПРИМЕЧАНИЕ. Эта настройка неприменима к следующим адаптерам:

  • Адаптер Intel® Wireless WiFi Link 4965AG_
  • Адаптер Intel® PRO/Wireless 3945ABG Network Connection

Защита смешанного режима

Используется для предотвращения конфликтов данных в смешанных сетевых средах 802.11b и 802.11g. Настройки RTS/CTS (Request to Send/Clear to Send) должны использоваться в средах, в которых клиенты могут не слышать друг друга. Значение CTS-to-self может использоваться для увеличения производительности в средах, в которых клиенты находятся близко и могут слышать друг друга.

Предпочитаемая частота

В среде с другими близлежащими излучающими радиоволны устройствами (например, микроволновые печи, беспроводные телефоны, точки доступа или клиентские устройства) в целях снижения уровня помех лучше использовать диапазон 5 ГГц вместо диапазона 2,4 ГГц и наоборот. Доступен следующий выбор:

  • без предпочтений;
  • диапазон 2,4 ГГц предпочтительней;
  • диапазон 5 ГГц предпочтительней.

Ниже представлен список различных диапазонов Wi-Fi:

  • устаревший 802.11 – 2,4 ГГц
  • 802.11a – 3,7 ГГц и 5 ГГц
  • 802.11b – 2,4 ГГц
  • 802.11g – 2,4 ГГц
  • 802.11n – 2,4 ГГц и 5 ГГц
  • 802.11ac – 5 ГГц

Агрессивность роуминга

Данный параметр позволяет определить агрессивность, с которой беспроводный клиент будет выполнять роуминг для поддержания подключения к точке доступа. Доступны пять настроек.

  • 3. Среднее: Эта настройка по умолчанию. Настройка баланса между отсутствием роуминга и производительностью.
  • 1. Мин.: Беспроводный клиент не выполняет роуминг. Только значительное ухудшение связи может вызвать поиск другой точки доступа.
  • 5. Макс.: Беспроводный клиент постоянно проверяет качество сигнала. Если произойдет снижение качества сигнала, он попытается найти и подключится к точке доступа с более сильным сигналом.

Мощность передачи

По умолчанию: Максимальная мощность.

Мин.: Минимальный диапазон работы: Настраивает адаптер на минимальную мощность передачи адаптера. Используется для дополнения числа областей распространения сигнала или ограничения области распространения. Снижает область работы адаптера в зонах интенсивного трафика для повышения общего качества передачи и предотвращения перегрузки и помех для других устройств.

Макс.: Максимальный диапазон работы: Настраивает адаптер на максимальную мощность передачи адаптера. Используется для установки максимальной производительности и диапазона работы в средах WiFi с ограниченным числом других радиопередающих устройств.

ПРИМЕЧАНИЕ. Оптимальной является установка, которая позволяет пользователю настроить мощность передачи на наиболее низкий уровень при поддержании приемлемого качества связи. Это обеспечивает работу максимального количества беспроводных устройств на небольших территориях и снижает образование помех для других устройств, использующих для передачи ту же частоту.

ПРИМЕЧАНИЕ. Этот параметр также работает в режимах сети (Infrastructure) или одноранговой сети (ad hoc).

Настройка "Wake on Magic Packet"

При условии использования этой настройки компьютер возобновляет работу после состояния сна после получения пакета "magic packet" от компьютера-отправителя. Пакет "magic packet" содержит MAC-адрес предназначенного целевого компьютера.

Включение приводит к применению настройки "Wake on Magic Packet". Отключение выключает применением настройки "Wake on Magic Packet". Отключение этой настройки приводит только к отключению этой функции "magic packet", но не функции "Wake on Wireless LAN".

Настройка "Wake on Pattern Match"

Эта функция приводит к возобновлению работы компьютера после состояния сна при получении определенного шаблона пробуждения на уровне адаптера. Функция поддерживается ОС Window* 7 и Windows 8. Типичные шаблоны:

  • пробуждение при новом входящем подключении TCP для протоколов IPv4 и IPv6 (TCP SYN IPv4 и TCP SYN IPv6);
  • пробуждение при получении пакетов повторной аутентификации стандарта 802.1x.

Отключение этой настройки приводит только к отключению функции совпадений шаблонов "pattern match", но не функции "Wake on Wireless LAN".

Режим беспроводной сети

Выберите режим, который будет использоваться для подключения к беспроводной сети:

  • Только 802.11a: Подключает адаптер WiFi только к сетям 802.11a. Доступно не для всех адаптеров.
  • Только 802.11b: Подключает адаптер WiFi только к сетям 802.11b. Доступно не для всех адаптеров.
  • Только 802.11g: Подключает адаптер WiFi только к сетям 802.11g.
  • 802.11a и 802.11g: Подключает адаптер WiFi только к сетям 802.11a и 802.11g. Доступно не для всех адаптеров.
  • 802.11b и 802.11g: Подключает адаптер WiFi только к сетям 802.11b и 802.11g. Доступно не для всех адаптеров.
  • 802.11a, 802.11b и 802.11g: (По умолчанию) - Выполняет подключение к беспроводным сетям 802.11a, 802.11b или 802.11g. Доступно не для всех адаптеров.
