アダプターの設定

[詳細設定] タブには、コンピューターにインストールされた WiFi アダプターのデバイス・プロパティーが表示されます。

アクセスするには

インテル(R) PROSet/Wireless WiFi 接続ユーティリティーの [詳細設定] メニューで [アダプターの設定] をクリックします。 [詳細設定] タブをクリックします。

WiFi アダプターの設定の説明

名前

 説明

802.11ac モード
(5 GHz)

802.11n 規格に基づいて策定された 802.11ac 規格。802.11ac モードは、チャネル帯域幅を 80MHz に増やし、より高密度の変調 (256 QAM) を追加することで、最大 867 Mbps (理論上) を実現します。[有効] または [無効] を選択して、WiFi アダプターの 802.11ac モードを設定します。デフォルトの設定は [有効] です。この設定は、802.11ac 対応アダプターのみに適用されます。

802.11n チャネル幅 (2.4 GHz)

パフォーマンスを最適化するように高スループットのチャネル幅を設定します。チャネル幅を [自動] または [20MHz] に設定します。802.11n チャネルが制限されている場合に [20MHz] を使用します。この設定は、802.11n 対応アダプターのみに適用されます。

注： この設定は、Intel® Wireless WiFi Link 4965AGN (20 MHz チャネル幅のみ使用) には適用されません。

802.11n チャネル幅 (5.2 GHz)

パフォーマンスを最適化するように高スループットのチャネル幅を設定します。チャネル幅を [自動] または [20MHz] に設定します。802.11n チャネルが制限されている場合に [20MHz] を使用します。この設定は、802.11n 対応アダプターのみに適用されます。

注： この設定は、次のアダプターには適用されません。

  • Intel(R) WiFi Link 1000
  • Intel(R) Wireless WiFi Link 4965AGN

802.11n モード

802.11n 標準は、既存の 802.11 標準に MIMO (Multiple Input Multiple Output) 技術を追加して策定されています。MIMO はデータのスループットを高めて、転送レートを向上させます。[有効] または [無効] を選択して、WiFi アダプターの 802.11n モードを設定します。デフォルトの設定は [有効] です。この設定は、802.11n 対応アダプターのみに適用されます。

注： 802.11n コネクションで 54 Mbps 以上の転送レートを得るには、WPA2*-AES セキュリティーを選択してください。セキュリティーを選択しない場合 (なし) でも、ネットワークのセットアップおよびトラブルシューティングを使用することができます。

管理者は、高スループット モードのサポートを有効または無効にして、電源使用量を減らしたり、他の帯域または互換性の問題による競合を軽減させたりできます。

アドホック チャネル 802.11b/g

[有効] または [無効] を選択します。

アドホック QoS モード

アドホック ネットワークのサービス品質 (QoS) 制御。QoS は、トラフィックの分類に基づいて、無線 LAN 経由でアクセス ポイントから送信されるトラフィックの優先付けを行います。WMM (Wi-Fi MultiMedia) は、Wi-Fi Alliance (WFA) の QoS 認証です。WMM を有効にすると、WiFi アダプターは WMM を使用して Wi-Fi ネットワークの優先度のタグ付けとキュー機能をサポートします。

  • WMM が有効 (デフォルト)
  • WMM を無効にする

Bluetooth® AMP

Bluetooth® AMP を有効または無効にします。AMP は alternate MAC/PHY の略で、802.11 (Wi-Fi) を高速トランスポートとして使用します。無効な場合は、Bluetooth HS はオフになります。

HT モード/VHT モード/無効

 この設定では、HT モード (高スループットモード) または VHT モード (超高スループットモード) のいずれかを選択するか、もしくは両方のモードを無効にします。HT モードは 802.11n との互換性をサポートしている一方、VHT モードは 802.11ac との互換性をサポートしています。

40Mhz チャネル幅非互換

この設定では、WiFi アダプターが 2.4 GHz バンドで 40 MHz チャネルを優先しないことをアクセス ポイントに通知します。デフォルト設定はオフ (無効) で、アダプターはこの通知を送信しません。アクセス ポイントが引き続き 40 MHz チャネルを使用し、802.11n チャネル幅 (2.4 GHz) が [AUTO] に設定されている場合は、WiFi アダプターも 40 MHz チャネルを使用します。

注： この設定は、次のアダプターには適用されません。

  • Intel(R) Wireless WiFi Link 4965AG_
  • Intel(R) PRO/Wireless 3945ABG Network Connection

混合モード保護

802.11b と 802.11g が混在する環境でのデータの衝突を避けるために使用します。クライアントが相互に認識できない環境では、Request to Send/Clear to Send (RTS/CTS) を使用します。クライアントが近接し相互に認識できる環境では、[個別 CTS 有効] を使用してスループットを向上できます。

優先バンド

その他の放射装置 (電子レンジ、コードレス電話機、アクセスポイント、クライアント デバイスなど) が近くにある環境では、干渉を減らすために、2.4GHz バンドで 5GHz バンドを優先したり、その逆を行うことができます。次の選択肢より選択します。

  • 優先なし
  • 2.4 GHz バンドを優先
  • 5 GHz バンドを優先

次に各種 Wi-Fi バンドを示します。

  • 802.11 レガシー - 2.4 GHz
  • 802.11a - 3.7 GHz および 5 GHz
  • 802.11b - 2.4 GHz
  • 802.11g - 2.40 GHz
  • 802.11n - 2.4 GHz および 5 GHz
  • 802.11ac - 5 GHz

ローミングの積極性

この設定では、アクセス ポイントへの接続を改善するためにワイヤレス クライアントをどれだけ積極的にローミングさせるかを指定します。5 種類の設定を使用できます。

  • 3. 中： これはデフォルトです。ローミングなしとパフォーマンスの間のバランスをとった設定となります。
  • 1. 最小： ワイヤレス クライアントはローミングしません。リンクの品質が極端に劣化したときのみ、別のアクセス ポイントへローミングします。
  • 5. 最大： ワイヤレス クライアントが継続的にリンクの品質を追跡します。リンクの品質が劣化した場合、ワイヤレス クライアントは品質の良いアクセス ポイントを参照し、ローミングを試行します。

送信電力

デフォルト設定： 最大パワー設定です。

最小： 最小範囲： アダプターの通信パワーを最小にします。範囲を広げることも、限定することもできます。高トラフィック領域内のカバレッジ エリアを縮小して、全体的な伝送品質を改善し、通信回線の過密化や他のデバイスとの干渉を防止できます。

最大： 最大範囲： アダプターの通信パワーを最大のレベルにします。追加の WiFi 無線装置数が限られている環境で最大のパフォーマンスとカバレッジ エリアを得るために選択します。

注： 最適な設定は、常にユーザーが通信の品質を犠牲にすることなく送信電力を最小レベルに設定することです。これによって、密度の高い領域で最大数のワイヤレス デバイスが稼動できるようになり、またこの無線と無線スペクトルを共有する他のデバイスとの干渉を軽減できます。

注： この設定は、ネットワーク (インフラストラクチャー通信) またはデバイスからデバイス (アドホック通信) モードを使用する場合に有効になります。

Wake on Magic Packet

この設定が有効な場合に、送信側コンピューターから「マジックパケット」を受信すると、コンピューターはスリープ状態からウェークアップします。マジックパケットには、対象の宛先コンピューターの MAC アドレスが含まれています。

有効にすると、Wake on Magic Packet はオンになります。無効にすると、Wake on Magic Packet はオフになります。これを無効にすると、マジックパケット機能が無効になるだけで、Wake on Wireless LAN は無効にはなりません。

Wake on Pattern Match

この機能は、アダプターで特定のウェークパターンが受信されたときに、コンピューターをスリープ状態からウェークアップさせます。この機能は Window* 7 および Windows* 8 でサポートされています。通常は次のパターンがあります。

  • IPv4 および IPv6の新しい着信 TCP 接続 (TCP SYN IPv4 および TCP SYN IPv6) によるにウェーク
  • 802.1x 再認証パケットによるウェーク

これを無効にすると、パターンマッチ機能が無効になるだけで、Wake on Wireless LAN は無効にはなりません。

ワイヤレス モード

ワイヤレス ネットワークへの接続に使用するモードを選択します。

  • 802.11a のみ： WiFi アダプターを 802.11a ネットワークのみに接続します。すべてのアダプターには適用されません。
  • 802.11b のみ： WiFi アダプターを 802.11b ネットワークのみに接続します。すべてのアダプターには適用されません。
  • 802.11g のみ： WiFi アダプターを 802.11g ネットワークのみに接続します。
  • 802.11a および 802.11g： WiFi アダプターを 802.11a および 802.11g ネットワークのみに接続します。すべてのアダプターには適用されません。
  • 802.11b および 802.11g： WiFi アダプターを 802.11b および 802.11g ネットワークのみに接続します。すべてのアダプターには適用されません。
  • 802.11a、802.11b、802.11g： (デフォルト) - 802.11a、802.11b、802.11g ワイヤレス ネットワークのいずれかに接続します。すべてのアダプターには適用されません。
