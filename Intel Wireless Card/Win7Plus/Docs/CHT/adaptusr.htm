介面卡設定

「進階」標籤會顯示安裝在您電腦上 WiFi 介面卡的裝置內容。

如何存取

在 Intel® PROSet/無線 WiFi 連線公用程式的進階功能表，按一下「進階設定」。選擇「進階」標籤。

WiFi 介面卡設定說明

名稱

 說明

802.11ac 模式
(5GHz)

802.11ac 標準根據 802.11n 標準建立。802.11ac 模式籍由提升通道頻寬至 80MHz 並加入更高密度的調變 (256 QAM)，最高可提供 867Mbps (理論值)。選取「啟用」或「停用」來設定 WiFi 介面卡的 802.11ac 模式。啟用是預設值。此設定僅適用於具有 802.11ac 功能的介面卡。

802.11n 頻寬寬度 (2.4 GHz)

設定高輸出量通道寬度以最大化效能。將通道寬度設定為「自動」或「20MHz」。在 802.11n 通道受到限制時使用 20MHz。此設定僅適用於具有 802.11n 功能的介面卡。

注意： 此設定不適用於 Intel® Wireless WiFi Link 4965AGN (僅使用 20 MHz 通道寬度)。

802.11n 頻寬寬度 (5.2 GHz)

設定高輸出量通道寬度以最大化效能。將通道寬度設定為「自動」或「20MHz」。在 802.11n 通道受到限制時使用 20MHz。此設定僅適用於具有 802.11n 功能的介面卡。

注意： 此設定不適用於下列介面卡：

  • Intel® WiFi Link 1000
  • Intel® Wireless WiFi Link 4965AGN

802.11n 模式

透過新增多重輸入多重輸出 (Multiple-Input Multiple-Output，MIMO)，802.11n 標準根據先前的 802.11 標準建立。MIMO 增加資料輸出量來提高傳輸速率。選取「啟用」或「停用」來設定 WiFi 介面卡的 802.11n 模式。啟用是預設值。此設定僅適用於具有 802.11n 功能的介面卡。

注意： 若要在 802.11n 連線上達成高於 54 Mbps 傳輸速率，必須選取 WPA2*-AES 保全性。可選取無保全性 (「無」) 來啟用網路設定和疑難排解。

管理員可啟用或停用對高輸出量模式的支援以減少電源消耗或與其他頻帶的衝突，或是減少相容性問題。

點對點 (ad hoc) 通道 802.11b/g

選取啟用或停用。

點對點 (ad hoc) QoS 模式

點對點網路中的服務品質 (QoS) 控制。QoS 透過基於流量分類的無線 LAN 從存取點提供流量的優先順序。WMM (Wi-Fi 多媒體) 是 Wi-Fi 聯盟 (WFA) 的 QoS 認證。啟用 WMM 後，WiFi 介面卡將使用 WMM 為 Wi-Fi 網路支援優先順序標籤和佇列功能。

  • WMM 啟用（預設）
  • WMM 停用

Bluetooth® AMP

啟用或停用 Bluetooth® AMP。AMP 是「Alternate MAC/PHY」(替代媒體存取控制/實體層) 之縮寫，採用 802.11 (Wi-Fi) 高速傳輸。如果停用，則高速藍芽亦一併關閉。

HT 模式/VHT 模式/已停用

 這項設定讓您選擇 HT 模式 (高輸出量模式)、VHT 模式 (極高輸出量模式)，或將兩種模式均停用。HT 模式支援 802.11n 相容功能，VHT 模式則支援 802.11ac 相容功能。

不容大量通道

這項設定會通知存取點，指出此 WiFi 介面卡在 2.4GHz 頻帶中不偏好使用 40MHz 通道。預設設定會讓此選項關閉 (停用)，如此介面卡就不會傳送這項通知。若存取點仍繼續使用 40MHz 通道，則只要 802.11n 頻寬 (2.4 GHz) 設定為「自動」，WiFi 介面卡也會使用 40MHz 通道。

注意： 此設定不適用於下列介面卡：

  • Intel® Wireless WiFi Link 4965AG_
  • Intel® PRO/Wireless 3945ABG Network Connection

混合模式保護

可以在混合 802.11b 及 802.11g 的環境中用來避免資料衝突。「要求傳送/等待傳送」(RTS/CTS) 應該用於用戶端彼此聽不到對方的環境中。「自我等待傳送」可以用在用戶端彼此鄰近並且可以聽到對方的環境中，用來提高輸出量。

偏好的頻帶

在一個周遭有其他放射裝置的環境中 (例如微波爐、無線電話、存取點或用戶端裝置)，為了減少干擾，您或許會想要優先採用 5GHz 頻帶而非 2.4GHz 頻帶，或是相反。您的選擇有：

  • 無偏好
  • 偏好 2.4GHz 頻帶
  • 偏好 5GHz 頻帶

以下為各種 Wi-Fi 頻帶：

  • 舊式 802.11 - 2.4GHz
  • 802.11a - 3.7GHz 及 5GHz
  • 802.11b - 2.4GHz
  • 802.11g - 2.4GHz
  • 802.11n - 2.4GHz 及 5GHz
  • 802.11ac - 5GHz

漫遊積極度

此設定讓您定義無線用戶端漫遊時，改良與存取點之間連線品質的積極程度。總共有五個設定可選。

  • 3. 中： 此為預設值。不漫遊與效能之間的平衡設定。
  • 1. 最低： 您的無線用戶端不會漫遊。只有在連線品質明顯下降時，才會漫遊到其他存取點。
  • 5. 最高： 您的無線用戶端不斷追蹤連線品質。只要品質下降，就嘗試尋找並漫遊到其他品質較好的存取點。

傳輸功率

預設設定： 最高的功率設定。

最低： 最小涵蓋範圍： 將介面卡設定在最低的傳輸功率。可讓您擴展涵蓋區域的數目或限制涵蓋區域。減少高流量區域的涵蓋範圍來改善整體傳輸品質，並避免擁擠和與其他裝置干擾。

最高： 最大涵蓋範圍： 將介面卡設定在最大的傳輸功率等級。在具有有限的其他 WiFi 無線電裝置的環境中選擇最大的效能和範圍。

注意： 最佳設定適用於始終以盡可能低且仍與通訊品質相容的級別設定傳輸功率的使用者。這樣允許在密度較大的區域使用最大數目的無線裝置，並減少與其共用的相同無線電展頻的其他裝置干擾。

注意： 當使用「網路（基礎架構）」或「點對點 (ad hoc)」模式時，此設定才會生效。

喚醒神奇封包

啟用此設定後，電腦從其他電腦收到「神奇封包」時即從睡眠狀態甦醒。神奇封包中含有預計目標電腦的 MAC 位址。

啟用即開啟「喚醒神奇封包」。停用即關閉「喚醒神奇封包」。停用此功能時，僅將停用神奇封包功能，不影響「無線區域網路喚醒」。

模式比對喚醒

在介面卡收到特定喚醒模式時，此功能會將電腦從睡眠狀態喚醒。Windows* 7 及 Windows 8 支援此功能。此類模式常見有：

  • IPv4 及 IPv6 新連入 TCP 連線喚醒 (TCP SYN IPv4 及 TCP SYN IPv6)。
  • 802.1x 重新驗證封包喚醒。

停用此功能時，僅將停用模式比對功能，不影響「無線區域網路喚醒」。

無線模式

選擇用於連線至無線網路的模式：

  • 僅限於 802.11a： 僅將無線 WiFi 介面卡連接到 802.11a 網路。並非適用於所有介面卡。
  • 僅限於 802.11b： 僅將無線 WiFi 介面卡連接到 802.11b 網路。並非適用於所有介面卡。
  • 僅限於 802.11g： 僅將無線 WiFi 介面卡連接到 802.11g 網路。
  • 802.11a 和 802.11g： 僅將 WiFi 介面卡連接到 802.11a 和 802.11g 網路。並非適用於所有介面卡。
  • 802.11b 和 802.11g： 僅將 WiFi 介面卡連接到 802.11b 和 802.11g 網路。並非適用於所有介面卡。
  • 802.11a、802.11b 和 802.11g： (預設) - 連接至 802.11a、802.11b 或 802.11g 無線網路。並非適用於所有介面卡。
