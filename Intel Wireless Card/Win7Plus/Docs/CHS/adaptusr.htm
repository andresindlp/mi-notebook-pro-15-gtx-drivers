适配器设置

高级选项卡显示安装在计算机上的 WiFi 适配器的设备属性。

如何访问

在英特尔® PROSet/无线 WiFi 连接实用程序的“高级”菜单，单击适配器设置。选择高级选项卡。

WiFi 适配器设置说明

名称

 说明

802.11ac 模式
(5GHz)

802.11ac 标准在 802.11n 标准的基础上构建。通过将信道带宽提升至 80MHz 并增加更高密度调制 (256 QAM)，802.11ac 模式将高达 867Mbps（从理论上讲）。选择启用或禁用设置 WiFi 适配器的 802.11ac 模式。“启用”为默认设置。此项设置仅适用于具有 802.11ac 功能的适配器。

802.11n 信道宽度 (2.4 GHz)

设定高吞吐量信道宽度设置以获取最佳性能。将信道宽度设为自动或 20MHz。如果 802.11n 信道受限制，则使用 20MHz。此项设置仅适用于具有 802.11n 功能的适配器。

注意： 此项设置不适用于 英特尔® Wireless WiFi Link 4965AGN（仅使用 20 MHz 信道宽度）。

802.11n 信道宽度 (5.2 GHz)

设定高吞吐量信道宽度设置以获取最佳性能。将信道宽度设为自动或 20MHz。如果 802.11n 信道受限制，则使用 20MHz。此项设置仅适用于具有 802.11n 功能的适配器。

注意： 此项设置不适用于以下适配器：

  • 英特尔® WiFi Link 1000
  • Intel® Wireless WiFi Link 4965AGN

802.11n 模式

802.11n 标准在先前的 802.11 标准基础上添加了多输入多输出 (MIMO)。MIMO 增加数据吞吐量以提高传输速率。选择启用或禁用设置 WiFi 适配器的 802.11n 模式。“启用”为默认设置。此项设置仅适用于具有 802.11n 功能的适配器。

注意： 要使 802.11n 连接上的传输速率高于 54 Mbps，必须选择 WPA2*-AES 安全性。无安全性（无）可被选择以启用网络设置和故障诊断。

管理员可以启用或禁用对高吞吐量模式的支持以减少耗电量和与其他频带的冲突或兼容性等问题。

Ad Hoc 信道 802.11b/g

选择启用或禁用。

Ad Hoc QoS 模式

Ad hoc 网络中的服务质量 (QoS) 控制。QoS 根据无线 LAN 上接入点的通信量提供通信量分类优先化。WMM (Wi-Fi Multimedia) 是 Wi-Fi Alliance (WFA) 的 QoS 认证。WMM 一经启用， WiFi 适配器便用它来支持 Wi-Fi 网络的优先级标记和排队功能。

  • WMM 被启用（默认）
  • WMM 被禁用

蓝牙® AMP

启用或禁用蓝牙® AMP。AMP 表示替代 MAC/PHY 并使用 802.11 (Wi-Fi) 进行高速传输。如果禁用，将关闭蓝牙 HS。

HT 模式/VHT 模式/禁用

 此项设置使您可以选择 HT 模式（高吞吐量模式）、VHT 模式（最高吞吐量模式）或禁用这两个模式。HT 模式支持 802.11n 兼容性，而 VHT 模式支持 802.11ac 兼容性。

快速信道耐受

此项设置与周围的接入点通信，告知它们此 WiFi 适配器不倾向于使用 2.4GHz 频带中的 40MHz 信道。默认设置是关闭（禁用），即适配器不发送此通知。如果接入点继续使用 40MHz 信道，WiFi 适配器将在 802.11n 信道带宽 (2.4GHz) 设置为“自动”时也使用 40MHz 信道。

注意： 此项设置不适用于以下适配器：

  • Intel® Wireless WiFi Link 4965AG_
  • 英特尔® PRO/无线 3945ABG 网络连接

混合模式保护

用于避免在 802.11b 和 802.11g 混用的环境中发生数据碰撞现象。请求发送/清除发送 (RTS/CTS) 应在客户端无法相互侦听到对方的环境中使用。在客户端靠近并且能互相侦听到对方的环境中，可使用 CTS-to-self 获取更大的吞吐量。

首选频带

在附近有其它辐射设备（如微波炉、无绳电话、接入点或客户端设备）时，为了降低干扰，您可能首选 5GHz 带宽，而非 2.4GHz 带宽，反之亦然。您的选择有：

  • 无首选项
  • 首选 2.4GHz 频带
  • 首选 5GHz 频带

此处是多种 Wi-Fi 带宽：

  • 802.11 传统型 - 2.4GHz
  • 802.11a - 3.7GHz 和 5GHz
  • 802.11b - 2.4GHz
  • 802.11g - 2.4GHz
  • 802.11n - 2.4GHz 和 5GHz
  • 802.11ac - 5GHz

漫游主动性

此设置让您定义您的无线客户端为改善与接入点连接的漫游主动程度。有五个可用设置。

  • 3. 中型： 这是默认情况。介于不漫游和性能之间的平衡的设置。
  • 1. 最低值： 您的无线客户端将不漫游。它仅在连接质量明显下降的情况下才会漫游到另一个接入点。
  • 5. 最高值： 您的无线客户端持续跟踪链接质量。一旦发生质量下降，则会试图寻找并漫游到一个较好的接入点。

传输功率

默认设置： 最高功率设置。

最低值： 最小覆盖面： 将适配器的传输功率设定为最低。使您能扩大覆盖区域数或限制一个覆盖区域。在高通讯量区域缩小覆盖范围可提高传输质量并避免阻塞现象和来自其他设备的干扰。

最高值： 最大覆盖面： 将适配器的传输功率设定为最高水平。选择此项以在额外 WiFi 无线电设备数目有限的环境中达到最佳性能和范围。

注意： 最佳设置使用户能始终将传输功率设为最低程度，而不影响其通信质量。这允许最大数目的无线设备在一个拥挤的区域操作，并减少来自此无线电与之共享无线电频谱的其他设备的干扰。

注意： 此项设置在使用“网络”（基础架构）模式或“设备到设备”(ad hoc) 模式时生效。

魔包唤醒

此项设置如果被启用，将在处于睡眠状态的计算机从发送方计算机接收到“魔包”时将其唤醒。魔包包括预定目的计算机的 MAC 地址。

启用将打开“魔包唤醒”。禁用将关闭“魔包唤醒”。禁用此项仅禁用魔包功能，不会禁用“无线 LAN 唤醒”。

模式唤醒匹配

当适配器接收到特别的唤醒模式时，此功能唤醒睡眠状态的计算机。Window* 7 和 Windows 8 支持此功能。此类模式通常为：

  • IPv4 和 IPv6（TCP SYN IPv4 和 TCP SYN IPv6）的新的传入 TCP 连接唤醒。
  • 802.1x 重新认证包唤醒。

禁用此项仅禁用模式匹配功能，不会禁用无线 LAN 唤醒。

无线模式

选择用来连接到无线网络的模式：

  • 仅 802.11a： 将无线 WiFi 适配器仅连接到 802.11a 网络。并非对所有适配器均适用。
  • 仅 802.11b： 将无线 WiFi 适配器仅连接到 802.11b 网络。并非对所有适配器均适用。
  • 仅 802.11g:将无线 WiFi 适配器仅连接到 802.11g 网络。
  • 802.11a 和 802.11g： 将 WiFi 适配器仅连接到 802.11a 和 802.11g 网络。并非对所有适配器均适用。
  • 802.11b 和 802.11g： 将 WiFi 适配器仅连接到 802.11b 和 802.11g 网络。并非对所有适配器均适用。
  • 802.11a、802.11b 和 802.11g： （默认） - 连接到 802.11a、802.11b 或 802.11g 无线网络。并非对所有适配器均适用。

确定

保存设置并返回前一页面。

取消

关闭并取消所有更改。
