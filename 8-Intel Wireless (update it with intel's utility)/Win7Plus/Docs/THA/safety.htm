ข้อมูลความปลอดภัย

เป็นสิ่งสำคัญที่คุณควรอ่านข้อมูลด้านความปลอดภัยเกี่ยวกับอแด็ปเตอร์ WiFi ของคุณ โปรดอ่าน คู่มือผู้ใช้ สำหรับการแจ้งข้อมูลเกี่ยวกับความปลอดภัยและระเบียบข้อบังคับ

ส่วนของยูทิลิตีการเชื่อมต่อ Intel® PROSet/Wireless WiFi ประกอบด้วยซอฟต์แวร์ภายใต้เงื่อนไขต่อไปนี้:

เปิดใบอนุญาตใช้งาน SSL

ลิขสิทธิ์ (c) 1998-2006 The OpenSSL Project. สงวนลิขสิทธิ์

อนุญาตให้เผยแพร่และใช้งานในรูปแบบต้นฉบับและไบนารี โดยมีหรือไม่มีการปรับเปลี่ยน ภายใต้เงื่อนไขต่อไปนี้:

  1. การเผยแพร่โค้ดต้นฉบับจะต้องมีคำประกาศลิขสิทธิ์ข้างต้น รวมถึงรายการเงื่อนไขนี้ และข้อสงวนสิทธิ์ที่ระบุไว้ด้านล่างนี้
  2. การเผยแพร่ในรูปแบบไบนารีจะต้องระบุคำประกาศลิขสิทธิ์ข้างต้น รวมถึงรายการเงื่อนไขนี้ และข้อสงวนสิทธิ์ที่ระบุไว้ด้านล่างนี้ ไว้ในเอกสารคู่มือ และ/หรือเอกสารอื่นๆ ที่จัดหาให้พร้อมกับการเผยแพร่ดังกล่าว
  3. เอกสารโฆษณาทั้งหมดที่ระบุคุณสมบัติหรือการใช้งานของซอฟต์แวร์นี้จะต้องแสดงข้อความต่อไปนี้: "ผลิตภัณฑ์นี้ประกอบด้วยซอฟต์แวร์ที่พัฒนาโดย OpenSSL Project สำหรับใช้งานใน OpenSSL Toolkit (http://www.openssl.org/)"
  4. ต้องไม่ใช้ชื่อ "OpenSSL Toolkit" และ "OpenSSL Project" เพื่อรับรองหรือส่งเสริมผลิตภัณฑ์ที่พัฒนาต่อยอดจากซอฟต์แวร์นี้ โดยไม่ได้รับอนุญาตเป็นลายลักษณ์อักษรล่วงหน้า สำหรับการอนุญาตเป็นลายลักษณ์อักษร โปรดติดต่อ openssl-core@openssl.org
  5. ผลิตภัณฑ์ที่พัฒนาต่อยอดจากซอฟต์แวร์นี้ต้องไม่ถูกเรียกว่า "OpenSSL" และ "OpenSSL" จะต้องไม่ปรากฏอยู่ในชื่อผลิตภัณฑ์ โดยไม่ได้รับอนุญาตเป็นลายลักษณ์อักษรล่วงหน้าจาก OpenSSL Project
  6. การเผยแพร่ในรูปแบบใดๆ จะต้องระบุข้อความต่อไปนี้: "ผลิตภัณฑ์นี้ประกอบด้วยซอฟต์แวร์ที่พัฒนาโดย OpenSSL Project สำหรับใช้งานใน OpenSSL Toolkit (http://www.openssl.org/)"

ซอฟต์แวร์นี้ได้รับการจัดหาโดย THE OpenSSL PROJECT "ตามสภาพที่เป็นอยู่" และไม่ว่าการรับประกันใดๆ ทั้งแบบแสดงอย่างชัดเจนหรือแบบโดยนัย รวมถึงแต่ไม่จำกัดเฉพาะการรับประกันความสามารถในการจำหน่ายต่อ และความเหมาะสมสำหรับการใช้เพื่อวัตถุประสงค์เฉพาะ ไม่ว่าในกรณีใดๆ THE OpenSSL PROJECT หรือผู้สนับสนุนจะไม่รับผิดชอบต่อความเสียหายโดยตรง ความเสียหายโดยอ้อม ความเสียหายที่เป็นอุบัติเหตุ ความเสียหายพิเศษ ความเสียหายที่เป็นตัวอย่าง หรือความเสียหายที่เป็นผลสืบเนื่อง (รวมถึงแต่ไม่จำกัดเฉพาะการจัดหาสินค้าหรือบริการทดแทน การสูญเสียการใช้งาน ข้อมูล หรือกำไร หรือการหยุดชะงักของธุรกิจ) ไม่ว่าจะมีสาเหตุจากอะไรก็ตาม และไม่ว่าจะใช้ทฤษฎีความรับผิดชอบใดก็ตาม ไม่ว่าจะเป็นความรับผิดชอบตามสัญญา ความรับผิดชอบแบบจำกัด หรือการละเมิด (รวมถึงการละเลยหรือกรณีอื่นใด) ที่เกิดขึ้นในลักษณะใดๆ จากการใช้ซอฟต์แวร์นี้ แม้กระทั่งในกรณีที่ได้รับการบอกกล่าวเกี่ยวกับความเสียหายที่อาจเกิดขึ้น

แพ็คเกจนี้เป็นการปรับใช้ SSL ที่พัฒนาโดย Eric Young (eay@cryptsoft.com) การปรับใช้นี้เป็นไปตาม SSL ของ Netscape

ไลบรารีนี้ปราศจากการใช้งานในเชิงพาณิชย์และไม่ใช่เชิงพาณิชย์ ตราบใดที่เป็นไปตามเงื่อนไขต่อไปนี้ เงื่อนไขต่อไปนี้ใช้กับโค้ดทั้งหมดที่พบในซอฟต์แวร์นี้ ไม่ว่าจะเป็น RC4, RSA, lhash, DES หรือโค้ดอื่นๆ ไม่ใช่เพียงแค่โค้ด SSL เท่านั้น เอกสารคู่มือ SSL ที่รวมอยู่ในซอฟต์แวร์นี้อยู่ภายใต้ข้อกำหนดลิขสิทธิ์เดียวกัน ยกเว้นข้อที่ว่าเจ้าของลิขสิทธิ์คือ Tim Hudson (tjh@cryptsoft.com)

ลิขสิทธิ์ยังคงเป็นของ Eric Young และด้วยเหตุนี้จะต้องไม่มีการเอาคำประกาศลิขสิทธิ์ใดๆ ในโค้ดออกไป หากแพ็คเกจนี้ถูกใช้ในผลิตภัณฑ์ Eric Young ควรจะได้รับการระบุว่าเป็นผู้พัฒนาส่วนต่างๆ ของไลบรารีที่ถูกใช้ โดยอาจอยู่ในรูปแบบของข้อความที่ปรากฏขึ้นเมื่อเริ่มต้นโปรแกรม หรือในเอกสารคู่มือ (แบบออนไลน์หรือหนังสือ) ที่ให้มาพร้อมกับแพ็คเกจ

อนุญาตให้เผยแพร่และใช้งานในรูปแบบต้นฉบับและไบนารี โดยมีหรือไม่มีการปรับเปลี่ยน ภายใต้เงื่อนไขต่อไปนี้:

  1. การเผยแพร่โค้ดต้นฉบับจะต้องมีคำประกาศลิขสิทธิ์ รวมถึงรายการเงื่อนไขนี้ และข้อสงวนสิทธิ์ที่ระบุไว้ด้านล่างนี้
  2. การเผยแพร่ในรูปแบบไบนารีจะต้องระบุคำประกาศลิขสิทธิ์ข้างต้น รวมถึงรายการเงื่อนไขนี้ และข้อสงวนสิทธิ์ที่ระบุไว้ด้านล่างนี้ ไว้ในเอกสารคู่มือ และ/หรือเอกสารอื่นๆ ที่จัดหาให้พร้อมกับการเผยแพร่ดังกล่าว
  3. เอกสารโฆษณาทั้งหมดที่ระบุคุณสมบัติหรือการใช้งานของซอฟต์แวร์นี้จะต้องแสดงข้อความต่อไปนี้: "ผลิตภัณฑ์นี้ประกอบด้วยซอฟต์แวร์การเข้ารหัสลับที่พัฒนาโดย Eric Young (eay@cryptsoft.com)" คำว่า 'การเข้ารหัสลับ' อาจถูกละเว้นหากคำสั่งรูทีนจากไลบรารีที่ถูกใช้งานไม่ได้เกี่ยวข้องกับการเข้ารหัสลับ
  4. หากคุณรวมโค้ดเฉพาะใดๆ ของ Windows (หรือผลงานพัฒนาต่อยอด) จากไดเรกทอรี apps (โค้ดของแอปพลิเคชัน) คุณจะต้องระบุข้อความต่อไปนี้: "ผลิตภัณฑ์นี้ประกอบด้วยซอฟต์แวร์ที่พัฒนาโดย Tim Hudson (tjh@cryptsoft.com)"

  3. ห้ามลบหรือเปลี่ยนแปลงคำประกาศนี้จากการเผยแพร่ต้นฉบับ

ซอฟต์แวร์นี้ได้รับการจัดหาโดยเจ้าของลิขสิทธิ์และผู้สนับสนุน "ตามสภาพที่เป็นอยู่" และไม่ว่าการรับประกันใดๆ ทั้งแบบแสดงอย่างชัดเจนหรือแบบโดยนัย รวมถึงแต่ไม่จำกัดเฉพาะการรับประกันความสามารถในการจำหน่ายต่อ และความเหมาะสมสำหรับการใช้เพื่อวัตถุประสงค์เฉพาะ ไม่ว่าในกรณีใดๆ เจ้าของลิขสิทธิ์หรือผู้สนับสนุนจะไม่รับผิดชอบต่อความเสียหายโดยตรง ความเสียหายโดยอ้อม ความเสียหายที่เป็นอุบัติเหตุ ความเสียหายพิเศษ ความเสียหายที่เป็นตัวอย่าง หรือความเสียหายที่เป็นผลสืบเนื่อง (รวมถึงแต่ไม่จำกัดเฉพาะการจัดหาสินค้าหรือบริการทดแทน การสูญเสียการใช้งาน ข้อมูล หรือกำไร หรือการหยุดชะงักของธุรกิจ) ไม่ว่าจะมีสาเหตุจากอะไรก็ตาม และไม่ว่าจะใช้ทฤษฎีความรับผิดชอบใดก็ตาม ไม่ว่าจะเป็นความรับผิดชอบตามสัญญา ความรับผิดชอบแบบจำกัด หรือการละเมิด (รวมถึงการละเลยหรือกรณีอื่นใด) ที่เกิดขึ้นในลักษณะใดๆ จากการใช้ซอฟต์แวร์นี้ แม้กระทั่งในกรณีที่ได้รับการบอกกล่าวเกี่ยวกับความเสียหายที่อาจเกิดขึ้น

อนุญาตให้เผยแพร่ซอฟต์แวร์นี้โดยอิสระ ในรูปแบบต้นฉบับและไบนารี (โดยมีหรือไม่มีการเปลี่ยนแปลง) ภายใต้เงื่อนไขต่อไปนี้:

  1. การเผยแพร่โค้ดต้นฉบับจะต้องมีคำประกาศลิขสิทธิ์ รวมถึงรายการเงื่อนไขนี้ และข้อสงวนสิทธิ์ที่ระบุไว้ด้านล่างนี้
  2. การเผยแพร่ในรูปแบบไบนารีจะต้องระบุคำประกาศลิขสิทธิ์ข้างต้น รวมถึงรายการเงื่อนไขนี้ และข้อสงวนสิทธิ์ที่ระบุไว้ด้านล่างนี้ ไว้ในเอกสารคู่มือ และ/หรือเอกสารอื่นๆ ที่เกี่ยวข้อง
  3. ไม่ใช้ชื่อของเจ้าของลิขสิทธิ์เพื่อรับรองผลิตภัณฑ์ที่สร้างขึ้นโดยใช้ซอฟต์แวร์นี้ หากไม่ได้รับอนุญาตเป็นลายลักษณ์อักษรอย่างเฉพาะเจาะจง
