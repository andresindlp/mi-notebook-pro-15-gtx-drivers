โหมด 802.11ac
(5GHz)

มาตรฐาน 802.11ac พัฒนาต่อจากมาตรฐาน 802.11n โหมด 802.11ac ให้ความเร็วถึง 867Mbps (ทางทฤษฎี) โดยการเพิ่มแบนด์วิดธ์ของแชนแนลเป็น 80MHz และเพิ่มความถี่ในการแปลงสัญญาณ (256 QAM) เลือก เปิดใช้งาน หรือ ปิดใช้งาน เพื่อตั้งโหมด 802.11ac ของอแด็ปเตอร์ WiFi โดย เปิดใช้งาน ถูกตั้งเป็นค่าเริ่มต้น การตั้งค่านี้ใช้กับอแด็ปเตอร์ที่รองรับ 802.11ac เท่านั้น

ขนาดแชนแนล 802.11n (2.4 GHz)

ตั้งขนาดแชนเนลสำหรับรับส่งข้อมูลไว้ที่ระดับสูงเพื่อเพิ่มประสิทธิภาพ ตั้งขนาดแชนเนลไว้ที่ อัตโนมัติ หรือ 20MHz ใช้ 20MHz ถ้าแชนเนล 802.11n ถูกจำกัด การตั้งค่านี้ใช้กับอแด็ปเตอร์ที่รองรับ 802.11n เท่านั้น

หมายเหตุ: การตั้งค่านี้ใช้ไม่ได้กับ Intel® Wireless WiFi Link 4965AGN (ใช้ขนาดแชนเนล 20 MHz เท่านั้น)

ขนาดแชนแนล 802.11n (5.2 GHz)

ตั้งขนาดแชนเนลสำหรับรับส่งข้อมูลไว้ที่ระดับสูงเพื่อเพิ่มประสิทธิภาพ ตั้งขนาดแชนเนลไว้ที่ อัตโนมัติ หรือ 20MHz ใช้ 20MHz ถ้าแชนเนล 802.11n ถูกจำกัด การตั้งค่านี้ใช้กับอแด็ปเตอร์ที่รองรับ 802.11n เท่านั้น

หมายเหตุ: การตั้งค่านี้ใช้ไม่ได้กับอแด็ปเตอร์ต่อไปนี้:

  • Intel® WiFi Link 1000
  • Intel® Wireless WiFi Link 4965AGN

โหมด 802.11n

มาตรฐาน 802.11n สร้างบนมาตรฐาน 802.11 ก่อนหน้านี้ โดยมีการเพิ่ม multiple-input multiple-output (MIMO) ทั้งนี้ MIMO จะเพิ่มปริมาณการรับส่งข้อมูลเพื่อปรับปรุงอัตราการถ่ายโอน เลือก เปิดใช้งาน หรือ ปิดใช้งาน เพื่อตั้งโหมด 802.11n ของอแด็ปเตอร์ WiFi โดย เปิดใช้งาน ถูกตั้งเป็นค่าเริ่มต้น การตั้งค่านี้ใช้กับอแด็ปเตอร์ที่รองรับ 802.11n เท่านั้น

หมายเหตุ: เพื่อให้อัตราการถ่ายโอนสูงกว่า 54 Mbps บนการเชื่อมต่อ 802.11n จะต้องเลือกความปลอดภัย WPA2*-AES สามารถเลือกไม่มีความปลอดภัย (ไม่มี) เพื่อเปิดใช้งานการตั้งค่าเครือข่ายและการแก้ไขปัญหา

ผู้ดูแลระบบสามารถเปิดใช้งานหรือปิดใช้งานการสนับสนุนสำหรับโหมดปริมาณการรับส่งข้อมูลในระดับสูง เพื่อลดการใช้พลังงานหรือปัญหาความขัดแย้งกับแบนด์อื่นๆ หรือปัญหาความเข้ากันได้

โหมด QoS เฉพาะกิจ

การควบคุม คุณภาพของการให้บริการ (QoS) ในเครือข่ายเฉพาะกิจ QoS จะจัดลำดับความสำคัญของการรับส่งข้อมูลจากจุดเชื่อมต่อสูงกว่า LAN ไร้สาย โดยขึ้นอยู่กับการจัดประเภทการรับส่งข้อมูล WMM (Wi-Fi Multimedia) เป็นการตรวจสอบ QoS ของ Wi-Fi Alliance (WFA) เมื่อ WMM ถูกเปิดใช้ อแด็ปเตอร์ WiFi จะใช้ WMM ในการสนับสนุนการใส่แท็กความสำคัญและการจัดลำดับความสามารถสำหรับเครือข่าย Wi-Fi

  • WMM เปิดใช้งานแล้ว (ค่าเริ่มต้น)
  • ปิดใช้งาน WMM

Bluetooth® AMP

เปิดใช้งานหรือปิดใช้งาน Bluetooth® AMP AMP ย่อมาจาก Alternate MAC/PHY และใช้ 802.11 (Wi-Fi) เป็นตัวขนส่งความเร็วสูง ถ้าปิดใช้งาน Bluetooth HS จะถูกปิด

โหมด HT/โหมด VHT/ปิดใช้งาน

 การตั้งค่านี้ทำให้คุณสามารถเลือกโหมด HT (โหมดอัตรารับส่งสูง) โหมด VHT (อัตรารับส่งสูงมาก) หรือปิดการใช้งานทั้งสองโหมด โหมด HT รองรับความเข้ากันได้ของ 802.11n ในขณะที่โหมด VHT รองรับความเข้ากันได้ของ 802.11ac

ไม่สามารถใช้กับแชนแนลขนาดใหญ่

การตั้งค่านี้สื่อสารกับจุดเชื่อมต่อเครือข่ายว่าอแด็ปเตอร์ WiFi นี้ไม่ชอบใช้ช่องสัญญาณ 40MHz ในแบนด์ 2.4GHz ค่าเริ่มต้นคือ ปิด (ปิดใช้งาน) ดังนั้นอแด็ปเตอร์จึงไม่ส่งการแจ้งเตือนนี้ ถ้าจุดเชื่อมต่อยังคงใช้ช่องสัญญาณ 40MHz ต่อไป อแด็ปเตอร์ WiFi ก็จะใช้ช่องสัญญาณ 40MHz ด้วย ถ้าตั้งค่าขนาดแชนแนล 802.11n (2.4GHz) เป็นอัตโนมัติ

หมายเหตุ: การตั้งค่านี้ใช้ไม่ได้กับอแด็ปเตอร์ต่อไปนี้:

  • Intel® Wireless WiFi Link 4965AG_
  • Intel® PRO/Wireless 3945ABG Network Connection

การป้องกันโหมดผสม

ใช้เพื่อหลีกเลี่ยงการชนกันของข้อมูลในสภาพแวดล้อมแบบผสมของ 802.11b และ 802.11g ฟังก์ชัน ขอให้ส่ง/ล้างเพื่อส่ง (RTS/CTS) ควรใช้ในสถานการณ์ที่เครื่องลูกข่ายอาจไม่สามารถรับสัญญาณระหว่างกันได้ CTS-to-self สามารถใช้เพื่อเพิ่มประสิทธิภาพการส่งผ่านมากขึ้นในสภาพแวดล้อมที่เครื่องลูกข่ายอยู่ในระยะใกล้เคียงกันและสามารถรับสัญญาณระหว่างกันได้

แบนด์ที่กำหนด

ในสภาพแวดล้อมที่มีอุปกรณ์อื่นๆ ที่ปล่อยพลังงานแม่เหล็กไฟฟ้าอยู่ใกล้เคียง (เช่นตู้อบไมโครเวฟ โทรศัพท์ไร้สาย จุดเชื่อมต่อเครือข่าย หรืออุปกรณ์ลูกข่าย) เพื่อลดการรบกวน จึงแนะนำให้สลับการใช้งานระหว่างแบนด์ 5GHz และ 2.4GHz ตัวเลือกได้แก่:

  • ไม่ชอบอย่างใดอย่างหนึ่ง
  • ชอบแบนด์ 2.4 GHz
  • ชอบแบนด์ 5 GHz

นี่คือแบนด์ Wi-Fi ต่างๆ:

  • 802.11 แบบเดิม - 2.4GHz
  • 802.11a - 3.7GHz และ 5GHz
  • 802.11b - 2.4GHz
  • 802.11g - 2.4GHz
  • 802.11n - 2.4GHz และ 5GHz
  • 802.11ac - 5GHz

ความแรงในการใช้งานข้ามเขต

การตั้งค่านี้จะช่วยคุณกำหนดว่าเครื่องลูกข่ายไร้สายจะข้ามเครือข่ายด้วยความแรงของสัญญาณมากน้อยเพียงใดเพื่อเพิ่มประสิทธิภาพการติดต่อกับจุดเชื่อมต่อ มีการตั้งค่าให้ใช้ห้าแบบ

  • 3. กลาง: นี่เป็นค่าเริ่มต้น ตั้งค่าให้สมดุลระหว่างการไม่ข้ามเครือข่ายกับประสิทธิภาพการเชื่อมต่อ
  • 1. ต่ำสุด: เครื่องลูกข่ายของคุณจะไม่มีการข้ามเครือข่าย เฉพาะกรณีที่คุณภาพการเชื่อมต่อต่ำลงอย่างชัดเจนเท่านั้นจึงจะมีการข้ามไปยังจุดเชื่อมต่ออื่น
  • 5. สูงสุด: เครื่องลูกข่ายไร้สายของคุณติดตามคุณภาพการเชื่อมต่ออย่างต่อเนื่อง หากคุณภาพสัญญาณลดลง เครื่องลูกข่ายก็จะพยายามค้นหาและข้ามเครือข่ายไปยังจุดเชื่อมต่อที่ดีกว่า

การส่งผ่านพลังงาน

การตั้งค่าเริ่มต้น: การตั้งค่ากำลังสูงสุด

ต่ำสุด: ระยะครอบคลุมต่ำสุด: ตั้งอแด็ปเตอร์ให้อยู่ที่ระดับกำลังการส่งต่ำสุด ทำให้คุณสามารถเพิ่มปริมาณพื้นที่ครอบคลุมหรือจำกัดขอบเขตพื้นที่ครอบคลุม ลดขนาดพื้นที่ครอบคลุมในบริเวณที่มีการใช้สัญญาณสูงเพื่อเพิ่มคุณภาพการส่งโดยรวม และหลีกเลี่ยงความหนาแน่นของการใช้สัญญาณและการเกิดสัญญาณรบกวนกับอุปกรณ์อื่นๆ

สูงสุด: ระยะครอบคลุมมากที่สุด: ตั้งอแด็ปเตอร์ให้อยู่ที่ระดับกำลังการส่งสูงสุด เลือกเพื่อให้มีประสิทธิภาพสูงสุดและอยู่ในสภาพแวดล้อมที่มีอุปกรณ์วิทยุ WiFi อื่นๆ ในปริมาณจำกัด

หมายเหตุ: การตั้งค่าดีที่สุดใช้สำหรับผู้ใช้ที่มักจะตั้งกำลังการส่งไว้ที่ระดับต่ำสุดเท่าที่เป็นไปได้โดยที่ยังสามารถใช้การสื่อสารอย่างมีคุณภาพได้ การตั้งค่านี้จะช่วยให้สามารถใช้อุปกรณ์ไร้สายได้จำนวนสูงสุดในพื้นที่ที่มีการใช้สัญญาณหนาแน่น และลดสัญญาณรบกวนกับอุปกรณ์อื่นที่วิทยุนี้ใช้ช่วงคลื่นวิทยุร่วมกัน

หมายเหตุ: การตั้งค่านี้จะมีผลเมื่อใช้โหมดเครือข่าย (โครงสร้างพื้นฐาน) หรือโหมดอุปกรณ์ต่ออุปกรณ์ (เฉพาะกิจ)

ปลุกด้วย Magic Packet

เมื่อเปิดใช้งานการตั้งค่านี้ คอมพิวเตอร์จะถูกปลุกจากสถานะสลีปเมื่อได้รับ "Magic Packet" จากคอมพิวเตอร์ที่ทำการส่ง Magic Packet ประกอบด้วยที่อยู่ MAC ของเครื่องคอมพิวเตอร์ปลายทาง

การเปิดใช้จะทำให้ปลุกด้วย Magic Packet ถูกเปิดทำงาน การปิดใช้งานจะเป็นการปิดการปลุกด้วย Magic Packet การปิดใช้งานจะเป็นการปิดเพียงคุณสมบัติ Magic Packet ไม่ใช่การปิดการปลุกด้วย Wireless LAN

ปลุกด้วย Pattern Match

คุณสมบัตินี้จะปลุกคอมพิวเตอร์จากสถานะสลีป เมื่ออแด็ปเตอร์ได้รับรูปแบบการปลุกแบบเฉพาะ คุณสมบัตินี้สนับสนุนโดย Window* 7 และ Windows 8 รูปแบบดังกล่าวมักเป็น:

  • ปลุกด้วยการเชื่อมต่อเข้ามาของ TCP สำหรับ IPv4 และ IPv6 (TCP SYN IPv4 และ TCP SYN IPv6)
  • ปลุกด้วยการตรวจสอบความถูกต้องอีกครั้งของกลุ่มข้อมูล 802.1x

การปิดใช้งานจะเป็นการปิดเพียงคุณสมบัติ Magic Packet ไม่ใช่การปิดการปลุกด้วย Wireless LAN

โหมดไร้สาย

เลือกโหมดที่จะใช้เชื่อมต่อกับเครือข่ายไร้สาย:

  • 802.11a เท่านั้น: เชื่อมต่ออแด็ปเตอร์ WiFi ไร้สายกับเครือข่าย 802.11a เท่านั้น ไม่มีผลบังคับใช้กับอแด็ปเตอร์ทั้งหมด
  • 802.11b เท่านั้น: เชื่อมต่ออแด็ปเตอร์ WiFi ไร้สายกับเครือข่าย 802.11b เท่านั้น ไม่มีผลบังคับใช้กับอแด็ปเตอร์ทั้งหมด
  • 802.11g เท่านั้น: เชื่อมต่ออแด็ปเตอร์ WiFi ไร้สายกับเครือข่าย 802.11g เท่านั้น
  • 802.11a และ 802.11g: เชื่อมต่ออแด็ปเตอร์ WiFi ไร้สายกับเครือข่าย 802.11a และ 802.11g เท่านั้น ไม่มีผลบังคับใช้กับอแด็ปเตอร์ทั้งหมด
  • 802.11b และ 802.11g: เชื่อมต่ออแด็ปเตอร์ WiFi ไร้สายกับเครือข่าย 802.11b และ 802.11g เท่านั้น ไม่มีผลบังคับใช้กับอแด็ปเตอร์ทั้งหมด
  • 802.11a, 802.11b และ 802.11g: (ค่าเริ่มต้น) - เชื่อมต่อกับเครือข่ายไร้สาย 802.11a, 802.11b หรือ 802.11g ไม่มีผลบังคับใช้กับอแด็ปเตอร์ทั้งหมด
