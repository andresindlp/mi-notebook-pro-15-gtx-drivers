取决于您的英特尔 WiFi 适配器型号，您的适配器与 802.11a、802.11b、802.11g 和 802.11n（草案 2.0）无线标准兼容。此适配器以 5GHz 或 2.4GHz 的频率和高达 450 Mbps 的数据速率操作，您现在可以在大型或小型环境中将您的计算机连接到现有的具备多个接入点的高速网络。您的 WiFi 适配器根据接入点位置和信号强度维持自动数据速率控制，以获得可能达到的最高速度连接。WiFi 连接实用程序可以轻而易举地管理您所有的无线网络连接。由 WiFi 连接实用程序设置的配置式以 802.1X 网络验证提供了增强的安全性手段。
