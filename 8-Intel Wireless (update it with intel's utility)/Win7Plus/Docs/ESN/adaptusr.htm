Opciones del adaptador

La ficha Avanzadas muestra las propiedades del dispositivo del adaptador WiFi instalado en su equipo.

Cómo acceder

En el menú Avanzadas de la Utilidad de conexión Intel® PROSet/Wireless WiFi, haga clic en Opciones del adaptador. Seleccione la ficha Avanzadas.

Descripción de las opciones del adaptador Wi-Fi

Nombre

 Descripción

Modo 802.11ac
(5 GHz)

El estándar 802.11ac se basa en el estándar 802.11n. El modo 802.11ac proporciona hasta 867 Mbps (teórico) al aumentar el ancho de banda de canal a 80 MHz y añadir una modulación de mayor densidad (256 QAM). Seleccione Activado o Desactivado para ajustar el modo 802.11ac del adaptador Wi-Fi. Activado es el valor predeterminado. Esta opción sólo se aplica a los adaptadores compatibles con 802.11ac.

Ancho del canal 802.11n (2,4 GHz)

Ajusta el ancho de canal de alta capacidad para maximizar el rendimiento. Ajuste el ancho de canal a Automático o 20 MHz. Utilice 20 MHz si los canales 802.11n están restringidos. Esta opción sólo se aplica a los adaptadores compatibles con 802.11n.

NOTA: Esta opción no se aplica al adaptador Intel® Wireless WiFi Link 4965AGN (sólo utiliza ancho de canal de 20 MHz).

Ancho del canal 802.11n (5,2 GHz)

Ajusta el ancho de canal de alta capacidad para maximizar el rendimiento. Ajuste el ancho de canal a Automático o 20 MHz. Utilice 20 MHz si los canales 802.11n están restringidos. Esta opción sólo se aplica a los adaptadores compatibles con 802.11n.

NOTA: Esta opción no se aplica a los adaptadores siguientes:

  • Intel® WiFi Link 1000
  • Intel® Wireless WiFi Link 4965AGN

Modo 802.11n

El estándar 802.11n se basa en estándares 802.11 anteriores agregando Múltiple entrada múltiple salida (MIMO). MIMO aumenta la transmisión de datos para mejorar la velocidad de transferencia. Seleccione Activado o Desactivado para ajustar el modo 802.11n del adaptador Wi-Fi. Activado es el valor predeterminado. Esta opción sólo se aplica a los adaptadores compatibles con 802.11n.

NOTA: Para alcanzar velocidades de transferencia superiores a los 54 Mbps en conexiones 802.11n, debe seleccionarse la seguridad WPA2*-AES. Se puede optar por no utilizar seguridad (Ninguna) para poder configurar la red y solucionar problemas en ella.

Un administrador puede activar o desactivar la compatibilidad para el modo de alta capacidad de proceso para reducir el consumo de energía o conflictos con otras bandas, o evitar problemas de compatibilidad.

Canal Ad Hoc 802.11b/g

Seleccione Activado o Desactivado.

Modo QoS Ad Hoc

Control de Calidad de servicio (QoS) en redes Ad Hoc. QoS permite establecer prioridades de tráfico en una red LAN inalámbrica desde el punto de acceso basándose en la clasificación del tráfico. WMM (Multimedia Wi-Fi) es la certificación QoS de la Wi-Fi Alliance (WFA). Cuando WMM está activada, el adaptador WiFi la utiliza para respaldar las funciones de etiquetado y puesta en cola por prioridad para redes WiFi.

  • WMM activada (valor predeterminado)
  • WMM desactivado

Bluetooth® AMP

Activar o desactivar Bluetooth® AMP. AMP significa Alternativa MAC/PHY y utiliza 802.11 (Wi-Fi) como transporte de alta velocidad. Si está desactivado, el Bluetooth HS se desactiva.

Modo HT/Modo VHT/Desactivado

 Estos valores le permiten seleccionar el modo HT (modo de alta capacidad de proceso), el modo VHT (modo de capacidad de proceso muy alta) o desactivar ambos modos. El modo HT es compatible con 802.11n, mientras que el modo VHT es compatible con 802.11ac.

No compatible con canal de 40 MHz

Esta opción comunica a los puntos de acceso que este adaptador WiFi no prefiere canales de 40 MHz en la banda de 2,4 GHz. El valor predeterminado es que esté apagado (desactivado), de forma que el adaptador no envíe esta notificación. Si el punto de acceso continúa utilizando canales de 40 MHz, el adaptador WiFi también utilizará canales de 40 MHz si el valor de ancho de canal 802.11n (2,4 GHz) es AUTOMÁTICO.

NOTA: Esta opción no se aplica a los adaptadores siguientes:

  • Intel® Wireless WiFi Link 4965AG_
  • Intel® PRO/Wireless 3945ABG Network Connection

Protección de modo mixto

Se utiliza para evitar colisiones de datos en un entorno mixto de 802.11b y 802.11g. Se debe utilizar Solicitud de envío/Listo para envío (RTS/CTS) en un entorno en el que los clientes no puedan escucharse entre sí. Se puede usar Activado para CTS-to-self para obtener mayor rendimiento en un entorno en el cual los clientes estén próximos y puedan escucharse entre sí.

Banda preferida

En un entorno donde haya dispositivos de radiación cercanos (como hornos microondas, teléfonos inalámbricos, puntos de acceso o dispositivos de clientes), para reducir las interferencias puede que prefiera la banda 5 GHz a la banda 2,4 GHz, y viceversa. Las opciones son:

  • Sin preferencias
  • Preferencia de banda 2,4 GHz
  • Preferencia de banda 5 GHz

Aquí están las diferentes bandas Wi-Fi:

  • Antigua 802.11: 2,4 GHz
  • 802.11a: 3,7 GHz y 5 GHz
  • 802.11b: 2,4 GHz
  • 802.11g: 2,4 GHz
  • 802.11n: 2,4 GHz y 5 GHz
  • 802.11ac: 5 GHz
  • 802.11ad: 60 GHz

Agresividad de itinerancia

Este valor le permite definir la agresividad con la que un cliente inalámbrico realiza la itinerancia para mejorar la conexión a un punto de acceso. Hay cinco valores disponibles.

  • 3. Medio: este es el valor predeterminado. Un valor equilibrado entre la no itinerancia y el rendimiento.
  • 1. Mínimo: su cliente inalámbrico no realizará itinerancia. Solamente una degradación significativa de la calidad del vínculo hace que itinere a otro punto de acceso.
  • 5. Máximo: el cliente inalámbrico realiza un seguimiento continuo de la calidad de los enlaces. Si se produce alguna degradación, intenta encontrar un punto de acceso mejor y definir la itinerancia.

Potencia de transmisión

Cobertura mínima: opción de potencia máxima.

Mínimo: Cobertura mínima: establezca el adaptador en el nivel mínimo de potencia de transmisión. Permite ampliar el número de áreas de cobertura o limitar un área de cobertura. Reduce el área de cobertura en áreas de tráfico intenso para mejorar la calidad global de transmisión y evita la congestión y la interferencia con otros dispositivos.

Máximo: Cobertura máxima: establezca el adaptador en el nivel máximo de potencia de transmisión. Seleccione el que ofrezca el máximo rendimiento y cobertura en entornos con escasez de otros dispositivos de radio WiFi.

NOTA: La configuración óptima es que el usuario establezca siempre la potencia de transmisión en el nivel más bajo posible que siga siendo compatible con la calidad que espera de la comunicación. Con ello se permite funcionar al máximo número de dispositivos inalámbricos en áreas densas y reducir las interferencias con otros dispositivos con los que este dispositivo comparta espectro de radio.

NOTE: Este valor surte efecto cuando se utiliza el modo de Red (infraestructura) o el modo entre dispositivos (Ad Hoc).

Reactivación por paquete mágico

Este valor reactiva el sistema de un estado de inactividad cuando recibe un "paquete mágico" de un sistema emisor. El paquete mágico contiene una dirección MAC del sistema de destino.

Al habilitarla, se activa la opción de reactivación por paquete mágico. Al deshabilitarla, se desactiva la opción de reactivación por paquete mágico. Si se deshabilita, sólo se desactiva la función de paquete mágico, pero no la función de reactivación por LAN inalámbricas.

Reactivación por coincidencia de patrones

Esta función reactiva el sistema de un estado de inactividad cuando se recibe un patrón de reactivación concreto en el adaptador. Esta función es compatible con Windows* 7 y Windows 8. Dichos patrones son normalmente:

  • Reactivación por nueva conexión TCP entrante para IPv4 y IPv6 (TCP SYN IPv4 y TCP SYN IPv6).
  • Reactivación por paquetes de reautenticación 802.1x.

Si se deshabilita, sólo se desactiva la función de coincidencia de patrones, pero no la función de reactivación por LAN inalámbricas.

Modo inalámbrico

Seleccione el modo que se va a utilizar para la conexión a una red inalámbrica:

  • Sólo 802.11a: conecta el adaptador Wi-Fi solamente a redes 802.11a. No aplicable a todos los adaptadores.
  • Sólo 802.11b: conecta el adaptador Wi-Fi solamente a redes 802.11b. No aplicable a todos los adaptadores.
  • Sólo 802.11g: conecta el adaptador Wi-Fi solamente a redes 802.11g.
  • 802.11a y 802.11g: conecta el adaptador WiFi solamente a redes 802.11a y 802.11g. No aplicable a todos los adaptadores.
  • 802.11b y 802.11g: conecta el adaptador WiFi solamente a redes 802.11b y 802.11g. No aplicable a todos los adaptadores.
  • 802.11a, 802.11b y 802.11g: (Predeterminado): conectar con las redes inalámbricas 802.11a, 802.11b o 802.11g. No aplicable a todos los adaptadores.
