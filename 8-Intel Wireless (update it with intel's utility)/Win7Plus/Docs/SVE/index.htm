Med nätverkskortet för WiFi kan du nå WiFi-nätverk, dela filer och/eller skrivare, och till och med dela din internetanslutning. Alla dessa funktioner kan utforskas med ett WiFi-nätverk i ditt hem eller på kontoret. Denna lösning för WiFi-nätverk är utformad för användning både i hemmet och på arbetet. Det går att lägga till ytterligare användare och funktioner allteftersom ditt nätverk växer och förändras.

Den här guiden innehåller grundläggande information om Intel-kort. Den innehåller information om flera kortegenskaper som du kan ange för att styra och förbättra prestanda för ditt kort med ditt specifika trådlösa nätverk och din specifika miljö. Med trådlösa kort från Intel® möjliggörs snabbanslutning utan sladdar för stationära och bärbara datorer.

Beroende på vilken modellbeteckning ditt Intel WiFi-kort har är det kompatibelt med de trådlösa standarderna 802.11a, 802.11b, 802.11g och 802.11n (utkast 2.0). Med frekvensen 5 GHz eller 2,4 GHz och dataöverföringshastigheter på upp till 450 Mbit/s kan du nu ansluta datorn till befintliga höghastighetsnätverk som använder flera åtkomstpunkter i stora eller små datormiljöer. Ditt WiFi-kort upprätthåller automatisk kontroll över datahastigheten enligt åtkomstpunktsplatsen och signalstyrkan för att uppnå snabbast möjliga anslutning. Alla dina trådlösa nätverksanslutningar kan lätt hanteras av WiFi-anslutningsverktyget. Profiler som skapas i WiFi-anslutningsprogrammet tillhandahåller förbättrade säkerhetsåtgärder med 802.1X-nätverksautentisering.

EEPROM och fast programvara).
