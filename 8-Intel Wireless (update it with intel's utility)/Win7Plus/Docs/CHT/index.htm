使用 WiFi 網路卡，您可以存取 WiFi 網路、共用檔案或印表機，或共用網際網路連線。所有這些功能可以在家中或辦公室使用 WiFi 網路探索。此 WiFi 網路解決方案專為家用與商用設計。隨著網路需要的增加與變更，可以新增其他用戶與功能。

本指南包含關於 Intel 介面卡的基本資訊。其中包括有關幾種介面卡內容的資訊，您可以設定這些內容以控制和加強介面卡在特定的無線網路和環境中的效能。Intel® 無線介面卡無需纜線即可讓桌上型電腦和筆記型電腦快速連線。

您的介面卡與 802.11a、802.11b、802.11g 和 802.11n 無線標準相容性取決於 Intel WiFi 介面卡的型號。以 5GHz 或 2.4GHz 頻率並以高達 450 Mbps 的資料速率操作，您現在可以將電腦連線到在大或小的環境內使用多重存取點的現有高速網路。您的 WiFi 介面卡會根據存取點位置和訊號強度保持自動的資料速率控制，以盡可能達成最快的連線。WiFi 連線公用程式可輕鬆管理您所有的無線網路連線。透過 WiFi 連線公用程式設定的設定檔，可為先進的保全性措施提供 802.1X 網路驗證。
